4. CSS Units

(1.) px (Pixels) : A fixed unit of measurement.
1px = one pixel on the screen (roughly a tiny dot).


.box {
  width: 300px;
  height: 100px;
}

Width → always 300px
Height → always 100px


---------------------------------------------

(2.) % (Percentage)
Takes a percentage of its parent element.

example:
<div class="parent">
  <div class="child"></div>
</div>



.parent{
   width:400px;
}

.child{
   width:50%;
}


-----------------------------

(3). em : Relative to the font-size of its parent.

Example:
.parent {
  font-size: 20px;
}

.child {
  font-size: 2em;
}

Calculation: 2 × 20px = 40px


----------------------------------------

(4). rem : Relative to the root (html) font-size
---------------------------------------------------


(5). vh (Viewport Height) : Percentage of browser height.

Example:
.hero {
  height: 100vh;
}


(6). vw (Viewport Width) :Percentage of browser width.

Example:
.banner {
  width: 50vw;
}

BOX SIZING
The CSS box-sizing property defines how to calculate the width and height of an element: should the calculation include padding and borders, or not.


Default Behaviour:

{
  box-sizing:content-box
}

width + padding + border = actual width of an element


height + padding + border = actual height of an element


The `box-sizing` property solves this problem.

{
  box-sizing: border-box
}

#  Inline : It takes only required with and Any height and width properties will have no effect.

Position:

1. Static : By Default all elements are statically positioned, top , left , right are not applied.

2. Relative :  relative to its normal position

3. absolute : positioned based on the nearest non-static positioned parent element

4. fixed :  take the fixed position of the html

5. sticky : stick to the defined position using top,left,bottom




What is Flexbox:
Flexbox Box model is a one dimensional layout model that provides a method to align and arrange element in a very effective ways ( ROWS and COlUMNS) - One at a time

===== Flexbox Container and Flex items =====
Flexbox Container: Its is a element that holds flex items, where flex items are DIRECT children of flex Container
Flex Items :  Items are the Direct children of container

flex container is created using
[ display:flex ]

Some important Properties"

=======FLEX CONTAINER PROPERTIES=======

flex-direction* : row,column,row-reverse,column-reverse
flex-wrap*:  to wrap the child element in screen width
justify-content* : main-axis (end,start,center,space-around,space-between)
align-items* : cross-axis (end,start,center)
